Olympic Rings
=============

In this project you will write a program that displays a design similar to the Olympic Rings. Your program will have about 13 lines of code and use the following functions (some more than once):

*	`ellipse()`  
*	`size()`  
*	`noFill()`  
*	`stroke()`  
*	`strokeWeight()`
*	Optional `smooth()`
